What is Pin Code?

Pin Code (Postal Index Number) is a numerical code used by the postal system in India to identify different regions, districts, and post offices. It helps in efficient sorting and delivery of mail.

When was Pin Code introduced?

Pin Code was introduced on August 15, 1972, by the Indian Postal Department. It replaced the earlier system of manual sorting of mail based on geographical locations.
